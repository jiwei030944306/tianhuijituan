## 🚀 快速开始

### 1. 启动后端服务
```bash
cd backend
python -m uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

### 2. 启动前端服务（新终端）
```bash
cd frontend
npm run dev
```

### 3. 访问应用
- 前端：http://localhost:5173
- 后端API：http://localhost:8000
- API文档：http://localhost:8000/docs

## 🎯 使用说明

### 上传试题
1. **选择分类信息**
   - 选择科目（如：数学、语文、英语等）
   - 选择学段（初中/高中）
   - 选择年级（根据学段自动联动）
   - 点击"下一步"按钮

2. **选择文件夹**
   - 点击"选择文件夹"按钮
   - 选择包含JSON文件和图片的文件夹
   - 查看文件统计信息（JSON文件、图片数量）
   - 点击"开始上传"

3. **查看上传结果**
   - 查看成功/失败统计
   - 查看导入的题目数量
   - 查看图片数量
   - 点击"继续上传"开始新的上传

### 查看题目
- **筛选题目**：使用5个筛选条件（科目/学段/年级/题型/难度）
- **重置筛选**：点击"重置筛选"按钮清空所有条件
- **刷新列表**：点击"刷新"按钮重新加载题目
- **查看详情**：
  - 题干内容
  - 题目图片（如有）
  - 选项列表（正确答案高亮显示）
  - 答案（绿色文字）
  - 解析（灰色文字）
  - 知识点标签

### 题目显示特性
- **难度标签**：简单（绿色）、中等（橙色）、困难（红色）
- **题型标签**：单选题、多选题、填空题、简答题
- **正确答案**：绿色背景高亮 + "正确答案"标签
- **图片显示**：最大200x150px，自动隐藏加载失败的图片
- **知识点标签**：蓝色标签显示在卡片底部
